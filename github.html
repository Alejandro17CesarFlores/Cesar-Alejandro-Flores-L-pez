¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo basada en la nube que permite a los desarrolladores gestionar proyectos y colaborar en el código de software utilizando el sistema de control de versiones Git. Es una herramienta ampliamente utilizada para el control de versiones y la colaboración en proyectos de software, tanto de código abierto como privados.
